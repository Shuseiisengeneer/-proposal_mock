Graph Coloring

~8 min read
3 practice problems
+30 XP

Graph coloring is one of the most beautiful and widely-studied problems in combinatorics. At its core, the question is deceptively simple: given a graph, can we assign colors to vertices such that no two adjacent vertices share the same color?

Definition
A proper coloring of graph G = (V, E) is a function c: V → {1, 2, …, k} such that for every edge (u,v) ∈ E, we have c(u) ≠ c(v).

The chromatic number χ(G) is the minimum k for which a proper k-coloring exists. Computing χ(G) is NP-hard in general — but for small graphs (or graphs with special structure), we can find it efficiently.

Why it matters

Graph coloring appears everywhere: scheduling problems (classes that share students can't be at the same time), register allocation in compilers (variables that are "live" simultaneously need different registers), and map coloring (regions sharing a border get different colors).

Four Color Theorem (1976)
Every planar graph is 4-colorable. That is, if a graph can be drawn in the plane without edge crossings, then χ(G) ≤ 4. The proof required computer assistance and remains one of the most famous results in mathematics.

The greedy algorithm

The simplest approach: process vertices in some order, and assign each vertex the smallest color not used by any of its already-colored neighbors.

def greedy_color(adj, n): color = [-1] * n for v in range(n): used = {color[u] for u in adj[v] if color[u] != -1} color[v] = next(c for c in range(n) if c not in used) return max(color) + 1

This is fast — O(V + E) — but not optimal. The number of colors used depends on the vertex ordering.

Quick check
What is the chromatic number χ(K₅) of the complete graph on 5 vertices?
3 — it can always be done with 3 colors
4 — by the four color theorem
5 — every vertex is adjacent to every other vertex
It depends on the vertex ordering
✓ Correct! In K₅, every pair of vertices is connected, so all 5 vertices must receive different colors. χ(Kₙ) = n for any complete graph.

Bipartite graphs are always 2-colorable

A graph is bipartite if its vertices can be split into two sets such that every edge crosses between the sets. Such graphs have χ(G) ≤ 2 — and a simple BFS coloring discovers whether a graph is bipartite in O(V + E).

Example
A cycle graph Cₙ is bipartite if and only if n is even. An even cycle alternates between two colors perfectly. An odd cycle (like a triangle C₃) requires 3 colors.
