Chromatic Number
Medium Graph Theory +40 XP

Given a simple undirected graph G = (V, E), find the chromatic number χ(G) — the minimum number of colors needed to color the vertices such that no two adjacent vertices share the same color.

For a complete graph K_n, every pair of distinct vertices is adjacent, requiring exactly n colors. Your task is to determine χ(G) for a general sparse graph using backtracking.

The graph is given as an adjacency list with n ≤ 10 vertices and m ≤ 20 edges.

Example 1
Input
4 6
0 1
0 2
0 3
1 2
1 3
2 3
Output
4
Example 2
Input
3 2
0 1
1 2
Output
2

Constraints

  • 1 ≤ n ≤ 10
  • 0 ≤ m ≤ n(n-1)/2
  • Graph is simple and undirected
  • Time limit: 2 seconds

import sys
from itertools import product

def chromatic_number(n, adj):
  def is_valid(coloring, k):
      for u in range(n):
        for v in adj[u]:
          if coloring[u] == coloring[v]:
            return False
    return True

  for k in range(1, n + 1):
    for coloring in product(range(k), repeat=n):
      if is_valid(coloring, k):
        return k
  return n

n, m = map(int, input().split())
adj = [[] for _ in range(n)]
for _ in range(m):
  u, v = map(int, input().split())
  adj[u].append(v)
  adj[v].append(u)

print(chromatic_number(n, adj))
