Chromatic Number
Medium Graph Theory

Given an undirected graph G = (V, E) with n vertices and m edges, find the chromatic number χ(G) — the minimum number of colors needed to color the vertices such that no two adjacent vertices share the same color.

For this problem, the graph is guaranteed to be planar and will have at most 8 vertices. Output the chromatic number.

Note: By the four color theorem, every planar graph has χ(G) ≤ 4. You do not need to prove this — but you may use it.
Examples
Input
4 4
0 1
1 2
2 3
3 0
Output
2
Input
3 3
0 1
1 2
2 0
Output
3
Constraints
  • 1 ≤ n ≤ 8
  • 0 ≤ m ≤ n(n−1)/2
  • The graph is undirected and has no self-loops
  • The graph is planar
  • Time limit: 1 second
Python 3.11
# Chromatic Number — brute force backtracking # n ≤ 8, so exhaustive search is feasible import sys from itertools import combinations def solve(): n, m = map(int, input().split()) adj = [[False] * n for _ in range(n)] for _ in range(m): u, v = map(int, input().split()) adj[u][v] = adj[v][u] = True def can_color(k): color = [-1] * n def bt(v): if v == n: return True used = {color[u] for u in range(n) if adj[v][u] and color[u] != -1} for c in range(k): if c not in used: color[v] = c if bt(v + 1): return True color[v] = -1 return False return bt(0) for k in range(1, 5): if can_color(k): print(k) return solve()
